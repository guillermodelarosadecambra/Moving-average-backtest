# Moving-average-backtest
Moving average crossover trading strategy backtest using Python.
This project implements a simple moving average crossover trading strategy using Python.

The strategy is tested on historical S&P 500 data using the SPY ETF.

## Features

- financial data download with yfinance
- moving average indicators
- trading signal generation
- vectorized backtesting
- Sharpe ratio calculation
- drawdown analysis

## Tools

Python  
pandas  
numpy  
matplotlib  
yfinance
